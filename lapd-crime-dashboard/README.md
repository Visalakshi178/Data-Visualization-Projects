# 🔍 LAPD Crime Data Dashboard

> Interactive Tableau dashboard analyzing Los Angeles crime data from 2020 to present, built as a group project for ADTA 5250 – Large Data Visualization.

## 📊 Overview

This dashboard provides a comprehensive visual analysis of LAPD crime data, enabling exploration of crime trends, geographic hotspots, victim demographics, and arrest patterns across Los Angeles.

## 📋 Dashboard Sheets

| Sheet | Description |
|-------|-------------|
| Bar Chart | Crime counts by category |
| Line Chart | Crime trends over time |
| Stacked Bar Chart | Crime breakdown by type and period |
| Top Crime Area | Highest crime areas in LA |
| Top Crime Type | Most frequent crime types |
| Total Arrests | Arrest summary statistics |
| Total Crimes | Overall crime volume metrics |
| Treemap | Crime distribution by area |

## 🔍 Key Insights

- Identified top crime areas and highest frequency crime types across LA districts
- Analyzed monthly and yearly crime trends from 2020 to present
- Explored victim demographics by descent and gender
- Mapped crime hotspots by area name and premise type
